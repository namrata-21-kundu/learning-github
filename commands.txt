git remote add origin <url>
(where origin is like a nickname for the github url)

git remote - v  //checking if it worked

git branch -M main  //push to github
(renames branch to main -> main is the default branch of github)

git push -u origin main
(-u sets origin main as default so next time just type git  push)

git push
